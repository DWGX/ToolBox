ToolBox: 徐俊皮看了都叫好的工具箱

灵活、扩展、优雅的模块化工具集合

功能概览

  • 模块化设计: ToolBox 采用模块化设计，开发者可以轻松地扩展新功能。
  • 自动化模块加载: 在启动时自动扫描指定文件夹中的所有模块。
  • 优雅的 UI 界面: 基于 PyQt6，界面美观且易于使用。

使用方法

安装

ToolBox 依赖于 PyQt6，可以使用以下命令安装：

pip install PyQt6

运行

确保您的模块文件已放入 Modules 文件夹中，然后在项目根目录下运行：

python <您的代码文件>.py

代码结构

1. MainPanel 类

MainPanel 类是整个应用程序的主界面。它管理着模块的加载、导航和显示。

__init__(self)

初始化主界面，包括导航和模块显示区域。

load_modules(self)

调用 ModuleManager.get_modules() 获取所有 QWidget 模块类的实例。

switch_module(self, index)

切换当前显示的模块，确保用户可以查看和操作所选择的工具。

2. ModuleManager 类

ModuleManager 负责管理模块的加载和实例化，以单例模式实现，确保只有一个模块管理器实例。

load_modules(self)

扫描模块文件夹以获取所有可用模块文件。

load_single_module(self, module_name, module_path)

加载单个模块文件，查找其中的 QWidget 子类并实例化。

get_modules(self)

提供已加载模块的列表供主界面使用。

设计理念与转折

  • 模块化与松耦合: 每个模块独立运行，易于添加或移除。
  • 自动化与动态加载: 使用 ModuleManager 动态加载模块。
  • 优雅与简洁的界面: 主界面提供简洁的导航和模块显示。
